# ✂️ Site Barbearia 

🔹 **Site institucional** para uma barbearia local, com agendamento online, galeria de serviços e informações de contato.  

## 🚀 Tecnologias Utilizadas  
- **HTML5** → Estrutura semântica e acessível.  
- **CSS3** → Design moderno e responsivo (mobile-first).  
- **JavaScript** → Interatividade (galeria, formulário de agendamento).  
- **[Opcional] React** → Se usado para componentes dinâmicos.  

## 📌 Funcionalidades Principais  
✅ **Página Inicial** com banner de promoções e call-to-action.  
✅ **Galeria de Serviços** (corte, barba, tratamentos) com preços.  
✅ **Agendamento Online** (formulário funcional com validação).  
✅ **Localização** (Google Maps embutido).  
✅ **Design Atraente** (cores masculinas, fonts modernas).
